# Product Images - Guide

### Using phpMyAdmin

1. Upload your image to `uploads/productos/` folder
2. Open phpMyAdmin
3. Navigate to your database → `productos` table
4. Find the product row you want to update
5. Edit the `imagen` column and enter just the filename (e.g., `martillo-acero.jpg`)
6. Save

### Image File Requirements

- **Maximum file size**: 5MB recommended
- **Formats**: JPG, PNG, GIF, WEBP
- **Naming convention**: Use lowercase, no spaces (use hyphens instead)
  - Good: `TaladroInalambrico.jpg`
  - Bad: `Taladro_Inalámbrico.JPG`
